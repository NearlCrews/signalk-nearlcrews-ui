# Support

Choose the channel that matches the request:

- Use [GitHub Discussions](https://github.com/NearlCrews/signalk-nearlcrews-ui/discussions) for usage questions, integration help, and design discussion.
- Use the bug report form for reproducible package defects.
- Use the feature request form for reusable component or contract proposals.
- Use a [private security advisory](https://github.com/NearlCrews/signalk-nearlcrews-ui/security/advisories/new) for vulnerabilities or sensitive reports.
- Use the [Signal K community](https://signalk.org/) for platform questions unrelated to this package.

Do not include access tokens, credentials, private vessel data, or unsanitized server output in any public report.
